Sorunun özeti — neden kopyalanmıyor?
Antigravity yanlış formatta ID üretiyor
Antigravity büyük ihtimalle champion adını ya da Riot'un protobuf ID'sini (635, 728...) kopyalamaya çalışıyor. Ama TFT client'ın beklediği format tamamen farklı — alfabetik sıradaki indeks numarası. Bu Community Dragon'dan çekilen özel bir JSON'a dayanıyor.
İki farklı format — ikisi de farklı amaçlar için
Format 1 — TFT Client Import (Team Planner)
TFT client'ın Team Planner'ına direkt yapıştırılabilen kısa hex kod. Riot'un resmi copy/paste formatı.
01 37 00 35 34 ... TFTSet16 ↑ ↑ ↑ ↑ ↑ ↑ ver ch1 ch2 ch3 ch4 ... set_id Örnek (9 şampiyon, sıfır item): 01 + [hex_index * 9] + TFTSet16
✓ TFT client'a import edilir ⚠ Champion index = alfabetik sıra
Format 2 — MetaTFT Team Builder URL (Protobuf)
metatft.com/team-builder'a link olarak açılan uzun base64. Pozisyon + item bilgisi içeriyor.
https://www.metatft.com/team-builder/_CgUI+wQQAAoMCNgF... ↑ Base64(Protobuf binary)
Site linki ✓ Pozisyon + item detayı içeriyor
Format 1 — Champion Index nasıl hesaplanır?
Kaynak: tftchampions-teamplanner.json (Community Dragon)
// JSON endpoint: https://raw.communitydragon.org/latest/plugins/rcp-be-lol-game-data/global/default/v1/tftchampions-teamplanner.json // Yapısı: { "TFTSet16": [ { "character_id": "TFT16_Ahri", ... }, { "character_id": "TFT16_AncientAlt",...}, ... ] } // Champion index = bu dizideki sırası (1-indexed) // character_id alfabetik sıralanmış // Ahri → TFT16_Ahri → sort sonrası kaçıncı? → o sayı hex'e çevrilir
Encoder kodu — Antigravity'ye yazılacak
// TeamCodeEncoder.ts const TEAMPLANNER_URL = 'https://raw.communitydragon.org/latest/plugins/' + 'rcp-be-lol-game-data/global/default/v1/' + 'tftchampions-teamplanner.json'; let championIndexMap: Map<string, number> | null = null; async function loadChampionIndex(): Promise<Map<string, number>> { if (championIndexMap) return championIndexMap; const res = await fetch(TEAMPLANNER_URL); const data = await res.json(); // Set 16 için aktif set key'ini bul const setKey = Object.keys(data).find(k => k.startsWith('TFTSet16')) ?? 'TFTSet16'; const champions: Array<{character_id: string}> = data[setKey]; // character_id'ye göre alfabetik sırala const sorted = [...champions].sort((a, b) => a.character_id.localeCompare(b.character_id) ); // 1-indexed map: 'TFT16_Ahri' → 3 (örnek) const map = new Map<string, number>(); sorted.forEach((c, i) => map.set(c.character_id, i + 1)); championIndexMap = map; return map; } export async function encodeTeamCode( characterIds: string[], // ['TFT16_Ahri', 'TFT16_Brand', ...] setId = 'TFTSet16' ): Promise<string> { const indexMap = await loadChampionIndex(); // Her champion için hex index'i üret const hexParts = characterIds.map(id => { const idx = indexMap.get(id); if (!idx) { console.warn('Unknown champion:', id); return '00'; // blank slot } return idx.toString(16).padStart(2, '0'); }); return '01' + hexParts.join('') + setId; } // KULLANIM (OverlayPanel'de): // const code = await encodeTeamCode([ // 'TFT16_Ahri', 'TFT16_Brand', 'TFT16_Camille', ... // ]); // await electronAPI.clipboard.writeText(code); // // Sonuç: '010314...TFTSet16'
MetaTFT comp verisi — character_id nereden geliyor?
MetaTFT API'sinden champion name → character_id mapping
MetaTFT comp verisi genellikle şampiyon adını döndürür ("Ahri", "Brand" gibi). Bunu TFT16_Ahri formatına çevirmek için ikinci bir map gerekiyor:
// Community Dragon ana TFT verisi: https://raw.communitydragon.org/latest/cdragon/tft/en_us.json // champions[].name = "Ahri", champions[].character_id = "TFT16_Ahri" // Veya daha basit yöntem: // tftchampions-teamplanner.json içindeki character_id'den name türet: // 'TFT16_Ahri' → name = 'Ahri' (prefix'i at, camelCase varsa koru) function nameToCharacterId(name: string): string { // 'Ahri' → 'TFT16_Ahri' // 'Kog Maw' → 'TFT16_KogMaw' (boşlukları kaldır) // 'Cho Gath' → 'TFT16_ChoGath' const clean = name.replace(/[\s']/g, ''); return 'TFT16_' + clean; // NOT: Bu heuristic çoğunda çalışır ama // bazı özel isimler için JSON'da manuel bakılmalı }
⚠ Heuristic yeterli değil — ✓ JSON'dan tam map yap
Antigravity'ye verilecek tam yönerge
1. tftchampions-teamplanner.json'dan Set 16 şampiyonlarını çek, character_id alanına göre alfabetik sırala, 1'den başlayarak indeksle → Map oluştur

2. cdragon/tft/en_us.json'dan şampiyon adı → character_id eşlemesi yap ("Ahri" → "TFT16_Ahri")

3. Overlay comp'undaki champion listesi için encodeTeamCode() çağır → 01[hex_indices]TFTSet16 formatı üret

4. Clipboard'a bu string'i yaz — TFT Team Planner'da Ctrl+V ile import çalışır

5. Version 02 formatı (item dahil) belgelenmemiş → v01 üret, item'lar zaten planner'da elle ekleniyor
