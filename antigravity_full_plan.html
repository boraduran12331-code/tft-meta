🔴 Kritik — Team Code kopyalama (hemen yapılacak)
Öncelik 1MetaTFT Team Code — Protobuf formatı çözüldü
URL'den reverse-engineer edildi. Format: Base64(Protobuf binary)
// Proto şeması (inferred): message TeamComp { repeated Unit units = 1; // her champion uint32 level = 3; // 21 = level 9 uint32 flags = 4; // set version flag } message Unit { uint32 champion_id = 1; // örn: 728 = Ahri uint32 hex_position = 2; // 0-27 board, 28+ bench bytes item_ids = 4; // packed varints: [144, 101, 136] }
Comp listesindeki her comp için bu format üretilip clipboard'a kopyalanacak → TFT içinde Ctrl+V ile direkt import edilebilir.
protobufjs encode champion_id → numeric ID map gerekli item_id → numeric ID map gerekli clipboard API
Öncelik 2Champion ID + Item ID mapping tabloları
MetaTFT, Riot'un internal numeric ID'lerini kullanıyor. İki kaynak:
// 1) Community Dragon (ücretsiz, güncel): https://raw.communitydragon.org/latest/cdragon/tft/en_us.json // champion: { "characterName": "TFT16_Ahri", "characterRecordID": 728 } // 2) Data Dragon TFT endpoint: https://ddragon.leagueoflegends.com/cdn/{version}/data/en_US/tft-champion.json
Her patch'te güncellenmesi için cache + auto-refresh sistemi kurulacak.
