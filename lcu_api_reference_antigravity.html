LCU API Nedir?
LCU (League Client Update), League of Legends istemcisinin 2017'de tamamen yenilenen modern versiyonudur. Electron tabanlıdır ve içinde tam bir HTTP + WebSocket sunucu çalıştırır — sadece 127.0.0.1'de, her açılışta rastgele port ile. Bu sunucuya doğrudan istek atabilirsin.

⚠ Resmi değil Riot resmi olarak desteklemez ama açıkça yasaklamıyor — topluluğun yıllardır kullandığı bir grey-area API. lcu.kebs.dev (incelediğin repo) güncel swagger dokümantasyonunu tutuyor, 26.05 client versiyonu itibarıyla aktif.
Bağlantı — Lockfile yöntemi (Mac dahil)
Lockfile nedir?
LoL client her açıldığında kurulum klasöründe lockfile adlı bir dosya oluşturur. İçinde 5 alan var, hepsi iki nokta ile ayrılmış:
LeagueClient:<pid>:<port>:<password>:https # Örnek: LeagueClient:13268:50674:Vu8kO3mZ9pQx2wYn:https
Client kapandığında bu dosya silinir. Var/yok → online/offline tespiti de buradan yapılır.
Mac'te lockfile yolu
# Varsayılan Mac kurulum yolları (değişebilir!): /Applications/League of Legends.app/Contents/LoL/lockfile # veya Riot Client yüklüyse: /Applications/Riot Client.app/Contents/LoL/lockfile # Güvenilir yöntem: pgrep ile PID bul, lsof ile yolu oku $ pgrep -x "LeagueClient" → PID al $ lsof -p <PID> | grep lockfile → tam yol
✓ Mac'te tamamen çalışır — lockfile mekanizması platform bağımsız.
Antigravity'deki mevcut LCUClient.ts için önerilen kod
import { execSync } from 'child_process'; import * as fs from 'fs'; import * as https from 'https'; export class LCUClient { private port: number | null = null; private password: string | null = null; private agent = new https.Agent({ rejectUnauthorized: false }); // self-signed cert // Yöntem 1: lockfile doğrudan oku private findLockfileByPath(): string | null { const paths = [ '/Applications/League of Legends.app/Contents/LoL/lockfile', `${process.env.HOME}/Applications/League of Legends.app/Contents/LoL/lockfile`, ]; return paths.find(p => fs.existsSync(p)) ?? null; } // Yöntem 2: process'ten bul (daha güvenilir) private findLockfileByProcess(): string | null { try { const pid = execSync('pgrep -x LeagueClient', { encoding: 'utf8' }).trim(); const lsof = execSync(`lsof -p ${pid} 2>/dev/null | grep lockfile`, { encoding: 'utf8' }); const match = lsof.match(/\s(\/[^\s]+lockfile)/); return match ? match[1] : null; } catch { return null; } } connect(): boolean { const lockPath = this.findLockfileByProcess() ?? this.findLockfileByPath(); if (!lockPath) return false; const content = fs.readFileSync(lockPath, 'utf8'); const [, , port, password] = content.split(':'); this.port = parseInt(port); this.password = password; return true; } get authHeader(): string { return 'Basic ' + Buffer.from(`riot:${this.password}`).toString('base64'); } async request<T>(path: string): Promise<T> { const res = await fetch(`https://127.0.0.1:${this.port}${path}`, { headers: { Authorization: this.authHeader }, // @ts-ignore — Node fetch'te agent desteği agent: this.agent, }); return res.json(); } }
TFT için kritik LCU endpointleri
Oyun fazı — en önemli endpoint
GET/lol-gameflow/v1/gameflow-phase
// Döndürdüğü değerler (TFT için önemli olanlar): "None" // Client açık ama oyun yok "Lobby" // Oda kuruldu "Matchmaking" // Eşleşme aranıyor "ReadyCheck" // "Kabul Et" ekranı "ChampSelect" // Şampiyon seçimi / TFT'de yok "InProgress" // OYUN İÇİNDE ← overlay'i göster "WaitingForStats" // Oyun bitti, skor tablosu yükleniyor "EndOfGame" // Maç sonu ekranı "PreEndOfGame" // Maç sonu öncesi
Bu endpoint'i her 2 saniyede bir poll et. "InProgress" → overlay göster, diğerleri → overlay gizle/lobby modu.

GET/lol-gameflow/v1/session
// Dönen objeden kritik alanlar: { "phase": "InProgress", "gameData": { "gameId": 12345678, "queue": { "id": 1090, "type": "NORMAL_TFT" }, // ← TFT mi kontrol et "playerChampionSelections": [...], "teamOne": [...], "teamTwo": [...] } }

WebSocket — polling yerine event-driven
Polling yerine WebSocket kullan (önerilen)
HTTP polling (her 2s istek) yerine LCU'nun WebSocket eventlerini dinlemek çok daha verimli.
import WebSocket from 'ws'; const ws = new WebSocket( `wss://127.0.0.1:${port}`, { headers: { Authorization: authHeader }, rejectUnauthorized: false, // self-signed cert } ); ws.on('open', () => { // Event'e abone ol (opcode 5 = Subscribe) ws.send(JSON.stringify([5, 'OnJsonApiEvent_lol-gameflow_v1_gameflow-phase'])); ws.send(JSON.stringify([5, 'OnJsonApiEvent_lol-gameflow_v1_session'])); ws.send(JSON.stringify([5, 'OnJsonApiEvent_lol-end-of-game_v1_eog-stats-block'])); }); ws.on('message', (data: Buffer) => { const [opcode, event, payload] = JSON.parse(data.toString()); if (event === 'OnJsonApiEvent_lol-gameflow_v1_gameflow-phase') { console.log('Game phase:', payload.data); // "InProgress" → overlay göster } });
Event ismi formatı: OnJsonApiEvent_ + endpoint yolundaki / yerine _
Mac'e özgü notlar
✓ Çalışır
LCU API tamamen cross-platform. Port/password mekanizması, lockfile formatı, tüm HTTP endpointler ve WebSocket Mac'te Windows ile birebir aynı çalışır.
⚠ SSL Sertifikası
Self-signed sertifika — her platformda rejectUnauthorized: false veya Riot'un root CA'sını trust etmek gerekir. Node.js fetch ile NODE_TLS_REJECT_UNAUTHORIZED=0 ortam değişkeni de işe yarar ama production'da güvenli değil.
⚠ Lockfile yolu değişken
Mac'te kurulum yolu kullanıcıya göre değişebilir (/Applications vs ~/Applications). Bu yüzden pgrep + lsof yöntemi her zaman sabit lockfile yolunu aramaktan daha güvenilir.
✗ TFT board state doğrudan yok
LCU API gerçek zamanlı TFT board state (hangi unitler hangi hexte, altın, HP) döndürmez. Bunlar için ya Riot'un Live Game API'si (farklı, port 2999) ya da ScreenCaptureKit + OCR gerekir.

Live Game API (port 2999) — TFT board için
LCU API ile karıştırma — bu farklı bir API
Oyun içindeyken 127.0.0.1:2999'da ayrı bir sunucu açılır. Gerçek zamanlı oyun verisi buradan gelir:
// Oyun içi tüm olaylar (goldfarming, ölüm vs): GET http://127.0.0.1:2999/liveclientdata/eventdata // Aktif oyuncular: GET http://127.0.0.1:2999/liveclientdata/playerlist // Oyunun stats: GET http://127.0.0.1:2999/liveclientdata/gamestats // → { "gameMode": "TFT", "gameTime": 487.3, ... }
⚠ Sınırlı TFT desteği Bu API LoL için tasarlanmış. TFT'de gamestats çalışır ama unit/item detayları gelmiyor. Port 2999 sadece oyun açıkken aktif — LCU'dan (port değişken) farklı.
