Antigravity AI · TFT Mac Companion — Yol Haritası

Teknoloji stacki
Electron + React TypeScript Riot TFT API MetaTFT scraper / API SQLite (yerel cache) Tesseract OCR (yedek) macOS Accessibility API ScreenCaptureKit node-notifier Zustand state

Mac için Overwolf yok — yerli Electron overlay şart. ScreenCaptureKit ile oyun penceresi üstüne transparan NSWindow yerleştirilir.

Faz 1 — Temel ~3–4 hafta · MVP
Altyapı
Transparan overlay penceresi
Electron + Swift bridge ile TFT penceresini detect et, üstüne click-through NSWindow aç. setIgnoreMouseEvents toggle.
Veri
Riot API entegrasyonu
TFT Match v1, Summoner v1, League v1. API key yönetimi, rate-limit queue, yerel SQLite cache.
Meta
MetaTFT comp verisi
metatft.com/comps endpoint reverse-engineer veya resmi API. Tier, avg place, unit listesi, BIS item parse.
Algılama
Oyun detect & lifecycle
NSWorkspace ile League/TFT proses takibi. Lobby → Loading → InGame → PostGame state machine.
UI
Comp listesi overlay
MetaTFT'den gelen S/A/B tier comps. Collapse/expand, sürüklenebilir panel, opacity slider.
Bildirim
Temel item bildirimi
Hedef comp'a göre "Bu item senin için BIS!" toast. node-notifier + macOS native banner.
Faz 2 — Oyun İçi Zeka ~4–5 hafta · Gerçek zamanlı asistan
Okuma
TFT Local API (LCU)
League Client Update API'si ile mevcut board, şampiyon havuzu, altın, seviye, streak okuma. Polling 2s.
Algılama
Board state reader
LCU yeterli değilse ScreenCaptureKit + Tesseract OCR fallback. Şampiyon, item, hex pozisyon parse.
Öneri
Akıllı comp önerisi
Mevcut board'a göre en uygun 3 comp öner. Elde olan unitler + econ'a göre pivot önerisi.
İtem
Akıllı item bildirimi
Elindeki komponentlere göre yapılabilecek BIS itemlar. "Shred al → Morello tamamlanır" gibi spesifik alert.
Scouting
Lobby scouting
Rakiplerin rank, top-4 rate, favori comp geçmişi. Riot API match history ile 20 oyun analizi.
Augment
Augment istatistikleri
Sunulan augmentlerin avg place, pick rate. Seçilen compa göre "Bu augment sana S tier" highlight.
Faz 3 — Trick Engine ~3–4 hafta · Rakipsiz özellik
Trick
Pozisyon trickler

Trick
Econ trickler

Trick

AI
Antigravity AI koç
Claude/GPT-4o entegrasyonu. "Şu an ne yapmalıyım?" sorusuna board + meta verisiyle anlık yanıt.
Meta
Patch delta bildirimi
Yeni patch'te hangi comp güçlendi/zayıfladı. "Bu setin yeni S tieri: X" push notification.
Trick
Carousel önceliklendirme
Carousel'de hangi item/unit alınmalı. Seçilen compa göre sıralı öncelik listesi overlay.
